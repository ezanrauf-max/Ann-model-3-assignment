# 📂 Projects

## 💳 1. Loan Approval Prediction using ANN

### 🎯 Objective

Predict loan approval status based on applicant financial information.

### 🛠 Features

* Applicant Income
* Loan Amount
* CIBIL Score
* Employment Status
* Education
* Asset Information

### 📊 Results

| Metric    | Value |
| --------- | ----- |
| Accuracy  | 96%   |
| Precision | 96%   |
| Recall    | 95%   |
| F1 Score  | 95%   |

### 🔗 Repository

```text
ANN-Loan-Approval-Prediction
```

## 🏥 2. Medical Insurance Cost Prediction using ANN

### 🎯 Objective

Estimate medical insurance charges using demographic and health-related information.

### 🛠 Features

* Age
* Gender
* BMI
* Number of Children
* Smoking Status
* Region

### 📊 Results

| Metric   | Value         |
| -------- | ------------- |
| MAE      | 2835.26       |
| MSE      | 21,067,194.71 |
| RMSE     | 4589.90       |
| R² Score | 86.43%        |

### 🔗 Repository

```text
ANN-Medical-Insurance-Cost-Prediction
```

## 🏦 3. Bank Customer Churn Prediction using ANN

### 🎯 Objective

Predict whether a customer will leave the bank based on account and demographic information.

### 🛠 Features

* Credit Score
* Country
* Gender
* Age
* Balance
* Number of Products
* Estimated Salary

### 📊 Results

| Metric    | Value |
| --------- | ----- |
| Accuracy  | 87%   |
| Precision | 76%   |
| Recall    | 46%   |
| F1 Score  | 57%   |

### 🛡 Overfitting Prevention

✅ Dropout Layers

✅ Early Stopping

✅ Validation Monitoring
